### Static toolbar
```react-snippet
normal-toolbar
```
```jsx
<Toolbar>
  <ToolbarRow>
    <ToolbarSection align="start">
      <ToolbarTitle>Title</ToolbarTitle>
    </ToolbarSection>
  </ToolbarRow>
</Toolbar>
```

### Fixed toolbar
```react-snippet
fixed-toolbar
```
```jsx
<Toolbar fixed>
  <ToolbarRow>
    <ToolbarSection align="start">
      <ToolbarTitle>Title</ToolbarTitle>
    </ToolbarSection>
  </ToolbarRow>
</Toolbar>

{/* Adjust top margin for fixed toolbar */}
<Content>
  ...
</Content>
```
``` Content ``` will produce the ```<main>``` tag by default, but you can override this behavior with property ```component```
```jsx
<Content component="div">
  ...
</Content>
```

### Sections
```react-snippet
sections
```
``` ToolbarSection ``` could be aligned at start, at center and at end of toolbar
```jsx
<Toolbar>
  <ToolbarRow>
    <ToolbarSection align="start">
      Start
    </ToolbarSection>
    <ToolbarSection>
      Center
    </ToolbarSection>
     <ToolbarSection align="end">
      End
    </ToolbarSection>
  </ToolbarRow>
</Toolbar>
